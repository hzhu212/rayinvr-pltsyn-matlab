使用方法：

1. 绘图：勾选树状列表中的射线组（可多选），然后点击 Apply 按钮。

2. 快捷绘图：双击树状列表中的一个射线组，将单独绘制该射线组（不受已勾选射线组的影响，也不会影响已勾选的射线组），适用于快速查看某个射线组的波形。

3. 缩放振幅：如果绘制的地震图中振幅过高、相互覆盖，或振幅过低、不便观察，便需要对振幅进行缩放。方法为：在点击 Apply 按钮之前，先点选树状列表中的某个射线组，观察到“Scale By”文字框中出现相应的射线组，此后的所有绘图便按照该射线组中的最高振幅进行缩放。

4. 彩色绘图：默认的情况下，绘制的图形均使用黑色。在绘图区域中右键，选择“Enable Color”，此后的所有绘图均会使用彩色。如想要继续使用黑色，只需再次在绘图区域中右键，选择“Disable Color”即可。

5. 导出绘图：在菜单栏中选择“File->Export Plot”，将会把当前绘图区域复制到一个新的 figure 窗口中。在新窗口中可执行任何 MATLAB 所支持的复制和导出操作。

6. 其他功能：
    1) 在树状列表中右键，选择菜单中的“Expand All”和“Collapse All”可分别展开或折叠全部树节点。
